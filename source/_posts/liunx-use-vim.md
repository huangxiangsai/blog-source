分屏多文件启动Vim

使用大写的参数**O**来垂直分屏。
使用参数**n**来表示分几屏:
```
vim -On file1 file2 ...
```

使用小写的o参数来水平分屏:

```
vim -on file1 file2 ...
```

## 关闭分屏
关闭当前窗口
 `Ctrl+w  c` 或者 `:q`

在只有一个窗口时， `Ctrl+w  c`将不起作用 ，只能用`:q`或者`Ctrl+w q`来关闭窗口

如果我们需要看同一文件的两个部分的内容， 文件又很长， 这时我们需要再打开一个相同的文件 ，我们可以使用`Ctrl+w s` 或 `Ctrl+w v`(一个是上下分屏，另一个是左右分屏)命令实现

当我们已经打开一个文件后，才想到需要查看另一文件 ，我们就可以使用如下命令：

垂直分屏: `:sp filename` 

水平分屏：`:vsp filename`
